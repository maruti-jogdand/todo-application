# todo-application
Todo Application in Golang


## Initialize package

1. go mod init github.com/maruti-jogdand/todo-application
2. go mod tidy

## Dependencies

1. Gorilla Mux Router - `go get -u github.com/gorilla/mux`
2. MongoDB Driver - `go get go.mongodb.org/mongo-driver/mongo`

## Run

go run main.go
